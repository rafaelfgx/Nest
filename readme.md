# NEST

![](https://github.com/rafaelfgx/Nest/actions/workflows/build.yaml/badge.svg)

API using Nest, JWT Authentication and Authorization, Swagger, Folder-by-Feature Structure.

## TECHNOLOGIES

* [Node](https://nodejs.org)
* [Nest](https://nestjs.com)

## RUN

<details>
<summary>Visual Studio Code</summary>

#### Prerequisites

* [Node](https://nodejs.org)
* [Nest](https://nestjs.com)
* [Visual Studio Code](https://code.visualstudio.com)
* [Visual Studio Code Node Debug](https://code.visualstudio.com/docs/nodejs/nodejs-debugging)

#### Steps

1. Open **source** directory in **Visual Studio Code**.
1. Execute **npm run start**.
4. Open <http://localhost:3000>.

</details>

<details>
<summary>Docker</summary>

#### Prerequisites

* [Docker](https://www.docker.com/get-started)

#### Steps

1. Execute **docker compose -p nest up --build -d** in **docker** directory.
2. Open <http://localhost:3000>.

</details>
